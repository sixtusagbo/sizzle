# sizzle example

A small app that demonstrates every variant of the [sizzle](https://github.com/sixtusagbo/sizzle) toast package: the success and error presets, custom icons, a title-only toast, and the `context.sizzle` extension.

## Run it

From this directory, with a device or emulator attached:

```bash
flutter run
```

Tap any button to fire its toast.
